# Snake Game

A classic Snake game implementation using vanilla JavaScript, HTML5, and CSS3.

## Features

- Keyboard controls (arrow keys)
- Score tracking with high score display
- Game timer
- Restart functionality
- Random snake and food positioning

## Installation
```bash
git clone https://github.com/Parnava-Ghosh/snake-game.git
cd snake-game
```

Open `index.html` in your browser to play.

## How to Play

- Use arrow keys to control the snake
- Eat food to grow and increase your score
- Avoid hitting the walls
- Press restart to play again

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)

## Game Mechanics

- **Movement**: Grid-based positioning with directional updates every 200ms
- **Collision Detection**: Wall boundary checking
- **Growth System**: Snake grows by one segment per food eaten
- **Scoring**: +1 point per food item
